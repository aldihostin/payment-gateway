API Documentation

QRIS Payment Gateway REST API

POST

Create QRIS Payment

/api/qris/create

Request Body

Parameter Type Required Description
amount number Required Payment amount in IDR (minimum 1)

Example Request

{ "amount": 50000 }

Success Response

200 OK
{ "status": true, "creator": "Created Using @krsna_081", "message": "QRIS berhasil dibuat", "data": { "idtransaksi": "QRIS-A1B2C3D4", "jumlah": 50000, "expired": "2025-01-07T12:35:00.000Z", "qrisString": "00020101021126...", "imageqris": { "url": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...", "format": "base64", "size": "512x512", "type": "image/png" }, "originalAmount": 50000, "wasAmountAdjusted": false, "amountAdjustment": 0 }, "timestamp": "2025-01-07T12:30:00.000Z" }

Error Response

400 Bad Request
{ "status": false, "creator": "Created Using @krsna_081", "message": "Jumlah harus diisi dan lebih dari 0" }
GET

Check Payment Status

/api/qris/status/{transactionId}

Path Parameters

Parameter Type Required Description
transactionId string Required Transaction ID from create response

Success Response

200 OK
{ "status": true, "creator": "Created Using @krsna_081", "message": "Transaction status retrieved", "data": { "idtransaksi": "QRIS-A1B2C3D4", "jumlah": 50000, "expired": "2025-01-07T12:35:00.000Z", "status": "success", "createdAt": "2025-01-07T12:30:00.000Z", "paidAt": "2025-01-07T12:32:15.000Z", "paymentDetails": { "type": "CR", "amount": "50000", "qris": "static" } } }

Status Values

Status Description
pending Payment is waiting for completion
success Payment completed successfully
expired Payment time has expired
cancelled Payment was cancelled

Cancel Payment

/api/qris/cancel/{transactionId}

Path Parameters

Parameter Type Required Description
transactionId string Required Transaction ID to cancel

Success Response

200 OK
{ "status": true, "creator": "Created Using @krsna_081", "message": "Transaction cancelled successfully", "data": { "idtransaksi": "QRIS-A1B2C3D4", "status": "cancelled", "cancelledAt": "2025-01-07T12:33:00.000Z" } }
POST

Telegram Notification

/api/qris/telegram-notify

Description

Internal endpoint for sending payment notifications to Telegram. Automatically called when payment is successful.

Request Body

Parameter Type Required Description
transactionId string Required Transaction ID
amount number Required Payment amount
paidAt string Optional Payment timestamp
